🚀 POUR DÉMARRER L'APPLICATION :

1. Ouvrir un terminal dans ce dossier (frontend-angular)

2. Taper : npm start

3. Ouvrir le navigateur : http://localhost:4200/login

4. Se connecter avec N'IMPORTE QUEL username et password
   Exemple : admin / admin

5. ✅ Vous êtes connecté !
