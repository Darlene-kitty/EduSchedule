# 🚀 EduSchedule - Commencez Ici

## Démarrage Rapide

### Windows
```powershell
.\start-dev.ps1
```

### Linux/Mac
```bash
chmod +x start-dev.sh
./start-dev.sh
```

### Accès
- **Frontend**: http://localhost:3000
- **API Gateway**: http://localhost:8080
- **Eureka**: http://localhost:8761
